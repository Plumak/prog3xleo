### Conceptos sobre Backtracking


## Backtracking es:

<p>Construir soluciones paso a paso probando posibilidades.</p>

## El algoritmo:

# El algoritmo:

1. prueba una decisión
2. avanza recursivamente
3. si no sirve, vuelve atrás
4. prueba otra.


## Cómo pensar un árbol de exploración
<p>

Cada nodo:
<ul>
    +representa un estado 
</ul>


Cada flecha:

<ul>
    +representa una decisión.
</ul>    
</p>
